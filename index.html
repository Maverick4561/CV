The HTML Boilerplate Code

The HTML Boilerplate code refers to the standard template code piece that is required and must be included in every HTML document. It can be obtained by simply typing in an Exclaimation mark (!) and hitting the Enter/Tab key.

Decoding the HTML Boilerplate line by line

First Line

The First line of the Boilerplate is !DOCTYPE html. It states that the following document is an HTML 5 document(the latest version of HTML) and that the browser must render it as HTML 5 doc. This line used to be much more complicated in case of HTML:4 version.

Second Line

The Second Line of boilerplate is the html tag. This tag marks the beginning and end of all HTML documents. It is a mandatory element that must be present in every HTML doc. All of the HTML's code is written in between the html tags.

The head tag

The head tag marks the beginning and ending of the Head element in a HTML document. Every HTML doc consists of a Head and a Body element. The head element contains all the webpage-related information such as the Title of the page or its Metadata, link to external stylesheets, script etc.

The title element

The title element is used to give a title to our Homepage. This title is visible on the "title bar" or the "tab bar" of a webpage.

The meta element

The meta element specifies some MetaData about our webpage. A Metadata is a "data about data" or essentially some more information about our webpage that tells the browser as well as the search engine on "HOW" exactly to render the page in the browser.

A meta tag is an empty tag, it just provides information about the webpage such as its author, description, keywords, character set etc. Some of the important attributes of meta tag are:

  1. meta charset="UTF-8"

    It is used to specify the character encoding system to be used for rendering the webpage by the browser. The most commonly used encoding system is Unicode or "UTF-8". It includes almost all the characters across all written languages of the world, hence ensuring correct encoding.

  2. meta name="description" content="text"

    This meta tag gives a brief description about our html code and our webpage. This description is then used by search engines such as Google Search to display it in their Search results.

  3. meta name="author" content="text" AND meta name="keywords" content="text"

    Used to specify the name of the Author of the webpage and the keywords pertaining to the webpage
